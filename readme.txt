== WHAT IT IS ==

The djg_cf plugin is very easy and useful component. Contact form that works without having to reload the page. It works based on jquery and ajax.
Requires jQuery on fronted!

Dependencies:
    - Wolf 0.7.3

== HOW TO USE IT ==

You will need to add this little code in your layout:

< ?php if (Plugin::isEnabled('djg_cf')) { djg_cf(); } ?>

== NOTES ==

See example on: <a href="http://antea.pl/kontakt.html">http://antea.pl/kontakt.html</a>
